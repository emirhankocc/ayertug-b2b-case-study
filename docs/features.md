# Features

## Dealer Management

- Multi-step dealer application
- Document upload
- Approval and rejection workflow
- Salesperson assignment
- Dealer credit-limit management

## Product Management

- Categories and brands
- Product variants
- Package and unit pricing
- VAT calculation
- Search and filtering

## Order Management

- Dealer-created orders
- Salesperson-created orders
- Approval workflow
- Shipment tracking
- Delivery confirmation

## Financial Management

- Account ledger
- Dealer debt tracking
- Collection records
- Credit-limit validation
- TRY and EUR display support

## Security

- JWT authentication
- Role-based authorization
- Dealer data isolation
- Input validation
- Protected file access
